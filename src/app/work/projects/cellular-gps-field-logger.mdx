## Overview

Rapid prototype using a Quectel cellular module to push **GNSS** and device metrics to the cloud.
A Python client renders map overlays, supports **signed downloads**, and CSV exports.

### Highlights
- UART driver + GNSS parser, low-power fixes
- Buffered MQTT messages with backoff/retry
- Python GUI for live route & signal-quality overlays

### Outcome
Improved outdoor validation workflows; easy correlation of **position, RF, and battery** during drive tests.
