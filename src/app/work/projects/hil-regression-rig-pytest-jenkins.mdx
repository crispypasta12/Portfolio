## Overview

Built a **PyTest-based** HIL framework with pluggable fixtures. The rig integrates a Keysight power analyzer
and an R&S GNSS simulator to validate **power states**, **connectivity**, and **navigation**.

### Highlights
- PyTest **fixtures** for BLE/Wi-Fi/Cellular
- Jenkins pipelines with matrices and flaky-test quarantine
- JSON artifacts + dashboards for triage

### Outcome
Regression time dropped from days to hours with a stable signal on intermittent defects.
